## 🏗️ Architecture

WPFBase is a modern WPF MVVM framework featuring:

- **Modern MVVM** with CommunityToolkit.Mvvm source generators
- **Dependency Injection** using Microsoft.Extensions.DependencyInjection
- **AvalonDock** for Visual Studio-style docking windows
- **Comprehensive Services** (Navigation, Dialog, Theme, etc.)
- **Validation Support** with both DataAnnotations and FluentValidation
- **Reactive Programming** with System.Reactive
